# walkindude/homebrew-tap

Homebrew tap for [walkindude](https://github.com/walkindude)'s tools.

Formulas are generated and pushed automatically by [GoReleaser](https://goreleaser.com/) on each release of the corresponding upstream repo. Do not edit `Formula/*.rb` by hand — changes will be overwritten on the next tag.

## Install

```bash
brew install walkindude/tap/<tool>
```

## Tools

- [**gosymdb**](https://github.com/walkindude/gosymdb) — Go symbol and call-graph database backed by SQLite.
